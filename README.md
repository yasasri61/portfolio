# Yasasri Uppuluri - Portfolio

A modern, responsive, and interactive portfolio website built with React, Vite, Tailwind CSS, and Material UI. This project showcases my skills, projects, certifications, and achievements with a sleek dark-themed UI and smooth GSAP animations.

-   **Modern Tech Stack**: Built with React 19 and Vite for blazing fast performance.
-   **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices.
-   **Dark Mode Default**: Sleek dark theme by default, with a toggle to switch to light mode.
-   **Interactive UI**:
    -   **GSAP Animations**: Smooth entrance animations, scroll triggers, and typing effects.
    -   **Material UI**: Polished components for cards, modals, buttons, and navigation.
    -   **Collapsible Sidebar**: Navigation sidebar that can be collapsed for more screen space.
-   **Single Page Navigation**: Smooth scrolling to different sections (About, Skills, Projects, etc.) within a single page.
-   **State Management**: Redux Toolkit used for managing theme and navigation state.
